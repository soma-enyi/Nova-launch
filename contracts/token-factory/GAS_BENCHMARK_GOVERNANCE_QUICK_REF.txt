Gas Benchmarking for Governance Flows - Quick Reference
========================================================

RUN COMMANDS
------------
# All governance benchmarks
cargo test bench_gov --test '*' -- --nocapture

# Comprehensive report (recommended)
cargo test bench_gov_comprehensive_report -- --nocapture

# Individual operations
cargo test bench_gov_transfer_admin -- --nocapture
cargo test bench_gov_pause -- --nocapture
cargo test bench_gov_unpause -- --nocapture
cargo test bench_gov_update_fees_single -- --nocapture
cargo test bench_gov_update_fees_both -- --nocapture
cargo test bench_gov_batch_update_admin -- --nocapture
cargo test bench_gov_set_clawback_enable -- --nocapture

# Comparative analysis
cargo test bench_gov_update_fees_comparison -- --nocapture
cargo test bench_gov_batch_vs_individual -- --nocapture

# Stress test
cargo test bench_gov_stress_rapid_changes -- --nocapture

OPERATIONS BENCHMARKED
----------------------
1. transfer_admin()       - Admin transfer
2. pause()                - Pause contract
3. unpause()              - Unpause contract
4. update_fees()          - Update fees (single/both)
5. batch_update_admin()   - Batch admin updates
6. set_clawback()         - Toggle clawback
7. is_paused()            - Read pause status

METRICS
-------
- Min:     Minimum CPU instructions
- Max:     Maximum CPU instructions
- Avg:     Average CPU instructions
- Median:  50th percentile
- P95:     95th percentile
- P99:     99th percentile
- StdDev:  Standard deviation

EFFICIENCY TARGETS
------------------
Batch savings:        > 20%
Fee update overhead:  < 20%
Read/Write ratio:     3-5x
Std deviation:        < 10% of average

TROUBLESHOOTING
---------------

High variability:     Run on dedicated hardware
Baseline warnings:    Update baseline constants
Regressions:          Review recent code changes
