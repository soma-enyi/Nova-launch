# Issue #417 Quick Reference

## Function Signature

```rust
pub fn queue_proposal(
    env: &Env,
    proposal_id: u64,
) -> Result<(), Error>
```

## Usage Example

```rust
// After voting period ends and proposal succeeds
let result = queue_proposal(&env, proposal_id);

match result {
    Ok(()) => {
        // Proposal queued successfully
        // Can now execute after ETA
    }
    Err(Error::QuorumNotMet) => {
        // Proposal didn't get enough votes
    }
    Err(Error::VotingEnded) => {
        // Voting period still ongoing
    }
    _ => {
        // Handle other errors
    }
}
```

## State Flow

```
Created -> Active -> Succeeded -> Queued -> Executed
                  |
                  -> Defeated (if votes_for <= votes_against)
```

## Quorum Logic

```rust
// Simple majority required
if votes_for > votes_against {
    // Quorum met
} else {
    // Quorum not met
}

// Abstain votes don't count for or against
```

## Error Codes

| Error | Code | Description |
|-------|------|-------------|
| `ProposalNotFound` | 34 | Proposal doesn't exist |
| `VotingNotStarted` | 35 | Voting hasn't begun |
| `VotingEnded` | 36 | Voting still ongoing |
| `QuorumNotMet` | 45 | Insufficient votes |
| `InvalidStateTransition` | 49 | Wrong state or already queued |
