## User Actions at Each Step

### Step 1: Welcome
- **Actions**: Next, Skip
- **Highlight**: None (centered modal)
- **Duration**: ~5 seconds

### Step 2: Connect Wallet
- **Actions**: Previous, Next, Skip
- **Highlight**: Connect Wallet button (top-right)
- **Duration**: ~10 seconds
- **Note**: User doesn't need to actually connect

### Step 3: Token Details
- **Actions**: Previous, Next, Skip
- **Highlight**: Token form area
- **Duration**: ~15 seconds
- **Note**: Explains form fields

### Step 4: Review & Deploy
- **Actions**: Previous, Next, Skip
- **Highlight**: Deploy Token button
- **Duration**: ~10 seconds
- **Note**: Explains fees and review process

### Step 5: View Token
- **Actions**: Previous, Next, Skip
- **Highlight**: None (centered modal)
- **Duration**: ~10 seconds
- **Note**: Explains post-deployment

### Step 6: Complete
- **Actions**: Finish
- **Highlight**: None (centered modal)
- **Duration**: Until user clicks Finish

## Responsive Behavior

### Desktop (>768px)
- Tooltip positioned relative to highlighted element
- Full-width progress indicator
- Larger text and spacing

### Mobile (<768px)
- Tooltip centered or positioned to avoid overflow
- Compact progress indicator
- Smaller text and spacing
- Touch-friendly button sizes

## Summary

The tutorial provides a smooth, guided onboarding experience that:
- Automatically starts for new users
- Highlights relevant UI elements
- Shows clear progress
- Allows skipping at any time
- Celebrates completion
- Remembers completion status
- Can be manually restarted

All flows are designed to be intuitive, accessible, and performant across all devices.
